By: PIXINVENT Team

Looking for an admin dashboard that was made with you in mind, look no further than Apex Admin. A powerful admin dashboard template built on Bootstrap 4. Apex is developer-friendly, rich with features and highly customizable. We’ve followed the highest industry standards to bring you the very best admin template that is not only fast and easy to use but highly scalable. Apex Admin is super flexible, powerful, clean & modern responsive bootstrap admin template with unlimited possibilities.

It includes different demos with Light/Dark/Transparent layouts, Vertical/Horizontal layouts and LTR/RTL option with organized folder structure, clean & commented code, 85+ pages, 30+ components, 50+ charts, 40+ advance cards and many more.

Apex Admin comes with 2 niche dashboards and provides searchable navigation, advance cards and incredible support. Apex Admin can be used for any type of web applications: Project Management, eCommerce backends, CRM, Analytics, Fitness or any custom admin panels.

It is powered with HTML 5, SASS, Bootstrap 4 & Javascript which looks great on Desktops, Tablets, and Mobile Devices. Apex Admin template comes with starter kit which will help developers to get started quickly.
